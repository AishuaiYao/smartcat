# SmartCat - AI图像采集小程序

基于微信小程序的智能图像采集与管理平台，支持设备连接、数据采集、云端存储与AI训练。

## 功能特性

- **设备管理** - 自动搜索、添加、管理ESP32设备
- **实时图像采集** - 通过TCP/UDP协议实时获取摄像头图像
- **数据集管理** - 本地存储(最多500张)与云端同步
- **云端存储** - 基于微信云开发的数据集管理
- **批量操作** - 支持批量上传、删除、下载
- **远程控制** - 实时遥控机器人移动采集数据

## 技术栈

- 微信小程序原生开发
- 微信云开发 (数据库、存储)
- TCP/UDP Socket 通信
- ESP32 设备通信协议

## 快速开始

### 1. 环境准备

- 微信开发者工具
- 开通微信云开发
- ESP32设备（固件见 `E:\AICleanBot\main\app_main.cpp`）

### 2. 配置云开发

1. 在开发者工具中开通云开发
2. 创建数据库集合 `dataset`
3. 设置集合权限为"所有用户可读写"
4. 修改 `app.js` 中的云环境ID

### 3. 运行项目

```bash
# 导入项目到微信开发者工具
# AppID: wxe43671e5bc6f25c1
```

## 使用说明

### 设备连接
1. 连接ESP32 WiFi热点
2. 首页点击➕添加设备
3. 点击设备卡片进入控制面板

### 数据采集
1. 点击"数据采集"进入实时画面
2. 控制机器人移动采集图像
3. 图片自动保存到本地缓存

### 云端管理
1. 切换到家庭WiFi/移动数据
2. 点击"打开相册" > "云端"
3. 选择图片上传/下载/删除

## 注意事项

- 数据采集时需连接设备WiFi
- 上传云端需切换到可访问互联网的网络
- 本地缓存最多500张图片(约10MB)
- 云端数据库需配置读写权限
